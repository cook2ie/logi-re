## 🧩 Overview

- 기존 로지텍 공식 사이트를 바탕으로 한 리디자인 프로젝트
- 미니멀한 레이아웃과 감각적인 비주얼 톤 중심으로 재구성
- 브랜드 키 컬러 **#00FDCF**를 적극 활용해 아이덴티티 강화
- 제품 중심 정보 구조를 보다 직관적으로 정리

---

## 🛠 Tech Stack

- HTML / CSS / JavaScript
- jQuery

---

## 👨‍💻 Key Points

- 기존 사이트의 안정적이고 정돈된 구조는 유지하면서, 전체적인 분위기를 보다 세련되고 감각적으로 재해석
- 불필요한 요소를 줄이고 메인 비주얼 중심으로 화면을 구성해 몰입감 강화
- 브랜드 컬러를 전반적인 UI 흐름에 반영하여 통일감 있는 화면 구성
- 정보 전달과 비주얼 톤의 균형을 고려해 퍼블리싱 진행

## 💡 What I Learned

- 브랜드 리디자인은 단순히 화면을 바꾸는 작업이 아니라, 기존 브랜드가 가진 성격을 유지하면서도 사용자가 느끼는 인상과 무드를 새롭게 설계하는 과정이라는 점을 배웠습니다.
- 브랜드 컬러를 포인트 요소에만 사용하는 것이 아니라, 레이아웃 전반에 일관되게 반영할 때 사이트의 정체성과 완성도가 더욱 분명해질 수 있다는 점을 경험했습니다.
- 제품 중심 사이트에서는 정보 구조와 비주얼 무드가 함께 정리될 때 화면의 전달력과 완성도가 높아진다는 점을 체감했습니다.
