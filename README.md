## 🧩 Overview

기존 사이트는 브랜드 신뢰감과 정보 전달력은 충분했지만,  
전체적인 인상에서 다소 단정하고 딱딱하게 느껴지는 부분이 있었습니다.

이번 리디자인에서는  
로지텍 특유의 기술적이고 정제된 브랜드 이미지를 유지하면서도,  
보다 미니멀한 레이아웃과 감각적인 분위기를 더해  
시각적으로 더 몰입감 있는 방향으로 재구성했습니다.

---

## 🎯 Design Concept

- 기존의 안정적이고 정돈된 구조는 유지
- 불필요한 요소를 줄여 더 미니멀한 화면 구성 적용
- 브랜드 키 컬러 **#00FDCF**를 적극 활용해 아이덴티티 강화
- 주변기기 브랜드 특유의 차갑고 딱딱한 인상을 보다 감성적으로 재해석

---

## 👨‍💻 Key Points

- 제품 중심의 레이아웃을 더 직관적으로 정리
- 브랜드 컬러를 전반적인 UI 흐름에 반영해 통일감 강화
- 메인 비주얼 중심의 화면 구성으로 몰입도 향상
- 정보 전달과 비주얼 톤의 균형을 고려한 퍼블리싱 진행

---

## 💡 What I Learned

이번 작업을 통해 브랜드 리디자인은  
단순히 화면을 예쁘게 바꾸는 것이 아니라,  
기존 브랜드가 가진 성격은 유지하면서도  
사용자가 느끼는 분위기와 첫인상을 새롭게 설계하는 과정이라는 점을 배웠습니다.

또한 브랜드 컬러를 포인트로만 사용하는 것이 아니라  
레이아웃 전반에 일관되게 녹여낼 때  
사이트의 정체성과 완성도가 더 분명해진다는 점을 경험했습니다.
