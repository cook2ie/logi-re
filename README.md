# 🖱 Logitech Redesign (Personal Project)

기존 로지텍 공식 사이트를 바탕으로 리디자인한 프로젝트입니다.  
브랜드가 가진 전문적이고 정돈된 인상은 유지하되, 전체적으로는 더 미니멀하고 감성적인 무드로 재해석하는 데 집중했습니다.

## 🧩 Overview

기존 사이트는 정보 전달력이 분명하고 안정적인 구조를 가지고 있었지만,  
전체적인 분위기에서 다소 단정하고 딱딱하게 느껴지는 부분이 있다고 생각했습니다.

그래서 이번 리디자인에서는  
로지텍 특유의 기술적이고 신뢰감 있는 브랜드 이미지에  
조금 더 미니멀한 레이아웃과 감성적인 톤을 더해  
보다 세련되고 몰입감 있는 방식으로 브랜드 경험을 재구성했습니다.

---

## 🎯 Design Direction

- 기존의 정돈된 브랜드 구조는 유지
- 불필요한 요소를 덜어낸 미니멀한 화면 구성
- 로지텍의 키 컬러 **#00FDCF**를 적극 활용한 브랜드 아이덴티티 강화
- 주변기기 브랜드 특유의 차가운 느낌을 줄이고, 더 감각적인 분위기로 재해석

---

## 👨‍💻 Key Points

- 제품 중심의 정보 구조를 더 직관적으로 정리
- 브랜드 컬러를 포인트가 아닌 전반적인 레이아웃 흐름에 녹여 통일감 강화
- 딱딱한 인상을 줄이고, 더 가볍고 세련된 비주얼 톤으로 개선
- 기능 소개와 브랜드 메시지가 자연스럽게 이어지도록 화면 흐름 구성

## 💡 What I Learned

단순히 화면을 예쁘게 바꾸는 것이 아니라,  
기존 브랜드가 가진 성격을 해치지 않으면서도  
사용자가 느끼는 분위기와 첫인상을 새롭게 만드는 과정이 중요하다는 점을 배웠습니다.

또한 브랜드 컬러를 부분적으로만 사용하는 것이 아니라  
전체 레이아웃 안에서 일관성 있게 활용할 때  
사이트의 인상이 훨씬 또렷해질 수 있다는 점을 경험했습니다.
